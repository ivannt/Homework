Homework
========

Exercises of Part -1



1. Familiarize yourself with:
Microsoft Visual Studio
Microsoft Developer Network (MSDN) Library Documentation
2. Find information about Console.WriteLine() method.
3. Create, compile and run a “Hello C#” console application.
4. Modify the application to print your name.
5. Write a program to print the numbers 1, 101 and 1001.
6. Install at home:
Microsoft Visual Studio (or Visual Studio Express)
7. Create console application that prints your first and last name.
8. Create a console application that prints the current date and time.
9. Create a console application that calculates and prints the square of the number 12345.
10. Write a program that prints the first 10 members of the sequence: 2, -3, 4, -5, 6, -7, ...
11. Provide a short list with information about the most popular programming languages. How do they differ from C#?
12. Describe the difference between C# and .NET Framework.
13. * Write a program to read your age from the console and print how old you will be after 10 years.
